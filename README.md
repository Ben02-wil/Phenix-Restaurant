# Phenix Bar & Restaurant

Bienvenue sur le site web de Phenix Bar & Restaurant ! Une destination gastronomique et de divertissement sophistiquée réunissant une cuisine raffinée, des cocktails artisanaux et des événements exclusifs.

## 🌟 À Propos de Phenix

Phenix est un bar et restaurant élégant offrant une expérience gastronomique unique dans une atmosphère raffinée et conviviale. Nous nous spécialisons dans :

- **Gastronomie Raffinée** - Créations culinaires soigneusement élaborées
- **Cocktails Signature** - Boissons artisanales premium
- **Événements en Direct** - Performances musicales et occasions spéciales
- **Ambiance Raffinée** - Cadre sophistiqué pour des moments inoubliables

## 📋 Fonctionnalités

- **Design Réactif** - Entièrement optimisé pour les appareils de bureau, tablettes et mobiles
- **Navigation Fluide** - Système de menu facile à utiliser avec défilement lisse
- **Interface Interactive** - Menu hamburger pour mobile, boutons intuitifs
- **Langue Française** - Adapté pour le public francophone
- **Mise en Page Multi-Sections** - Accueil, Services, Menu, Événements et sections Contact

## 🛠️ Structure du Projet

```
phenix/
├── index.html       # Fichier HTML principal avec la structure de la page
├── script.js        # Fonctionnalité JavaScript (basculement menu, défilement lisse)
├── styles.css       # Styles complets et design réactif
├── images/          # Répertoire des ressources d'images
└── README.md        # Ce fichier
```

## 📄 Vue d'Ensemble des Fichiers

### `index.html`
Contient la structure HTML complète incluant :
- En-tête avec navigation et logo
- Section héros avec boutons d'appel à l'action
- Sections de services, menu, événements et contact
- Menu hamburger réactif pour mobile

### `script.js`
Gère la fonctionnalité JavaScript :
- Fonctionnalité de basculement du menu
- Navigation avec défilement lisse
- Boutons de réservation et découverte du menu
- Fermeture du menu au clic externe

### `styles.css`
Fournit des styles complets :
- Mises en page réactives avec grille et flexbox
- Palette de couleurs moderne et typographie
- Design réactif mobile-first
- Transitions et animations fluides
- Navigation d'en-tête collante

## 🌐 Navigation

Le site web comprend les sections principales suivantes :

1. **Accueil** - Bienvenue et présentation
2. **Services** - Services de restauration disponibles
3. **Menu** - Offres alimentaires et boissons
4. **Événements** - Événements spéciaux et concerts

## 🚀 Démarrage

1. Ouvrez `index.html` dans un navigateur web
2. Naviguez en utilisant la barre de menu ou les boutons
3. Cliquez sur « Réserver » pour faire une réservation
4. Explorez différentes sections en utilisant le défilement lisse

## 💾 Comment Utiliser

### Développement Local
- Ouvrez simplement le fichier `index.html` dans votre navigateur web
- Aucun outil de construction ou dépendances requis
- Tous les fichiers sont autonomes

### Déploiement
- Téléchargez tous les fichiers sur votre serveur d'hébergement web
- Maintenez la structure des répertoires (gardez le dossier `images/`)
- Assurez-vous que les fichiers `.html`, `.js` et `.css` se trouvent dans le répertoire racine

## ✨ Implémentation des Fonctionnalités Clés

### Menu Réactif
Le menu hamburger s'active automatiquement sur les petits écrans et bascule la visibilité au clic.

### Défilement Lisse
Tous les liens de navigation utilisent le comportement de défilement lisse pour une expérience moderne et raffinée.

### Boutons d'Appel à l'Action
- **Réserver une Table** - Dirige vers la section de réservation
- **Découvrir le Menu** - Fait défiler jusqu'à la section menu
